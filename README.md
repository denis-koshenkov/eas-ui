# Environemnt Alert System User Interface
GUI application to interact with environment alert system device over Bluetooth

# Installation
1. Install dependencies:
```
sudo apt install libxcb-cursor0
```

1. Create a python venv:
```
python -m venv env
```

1. Activate the venv:
```
source env/bin/activate
```

1. Install pyside6:
```
pip install pyside6
```

# Running the UI
Run from the root directory of this repo:
```
pyside6-project run ui
```
